# Переменные среды

## Общее  
```sh
# окружение - production, test или development
NODE_ENV  
```

## Приложение
```sh
# url сервера приложения 
APP_URL
# порт сервера приложения
APP_PORT
```

## API сервер
```sh
# порт api сервера
SERVER_PORT
# Включает логирование sql запросов
SEQUELIZE_LOGGING  
```

## СУБД
```sh
# хост БД
DB_HOST=localhost
# имя БД
DB_NAME
# пользователь
DB_USER
# пароль
DB_PASSWORD
```
